# 决策树产品识别系统

基于 FastAPI 和决策树算法的产品智能识别系统，通过交互式问答引导用户完成产品识别。

## 功能特性

- **智能决策树**：使用信息增益算法自动构建决策树
- **交互式问答**：通过 Web 界面引导用户回答问题
- **历史回退**：支持返回上一步修改答案
- **数据驱动**：从 Excel 文件加载产品特征数据
- **自动排序**：选项按数值、文本、缺失值智能排序

## 技术栈

- **后端**：FastAPI + Uvicorn
- **模板引擎**：Jinja2
- **数据处理**：Pandas + NumPy
- **决策算法**：ID3 决策树（信息增益）

## 项目结构

```
.
```

## 快速开始

### 1. 安装依赖

```bash
# 使用 uv（推荐）
uv sync

# 或使用 pip
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python main.py
```

或使用 uvicorn：

```bash
uvicorn main:app --reload
```

### 3. 访问应用

打开浏览器访问 http://127.0.0.1:8000

## 数据格式

`output.xlsx` 文件格式要求：

- **第一列**：产品型号（目标分类）
- **第二、三列**：排除列（不用于决策）
- **后续列**：产品特征属性

示例：

| 型号 | 品牌 | 类别 | 颜色 | 尺寸 | 材质 |
|------|------|------|------|------|------|
| A001 | XX | 电子 | 黑色 | 10寸 | 金属 |
| A002 | XX | 电子 | 白色 | 12寸 | 塑料 |

## 工作原理

1. **数据加载**：从 Excel 读取产品数据
2. **树构建**：使用 ID3 算法基于信息增益构建决策树
3. **交互流程**：
   - 用户点击"开始识别"
   - 系统根据当前节点显示问题
   - 用户选择答案后进入下一节点
   - 到达叶子节点时显示识别结果
4. **回退功能**：可返回上一步重新选择
